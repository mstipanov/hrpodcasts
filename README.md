# HR Podcasts

This simple GoogleAppEngine app can be used to generate RSS feeds for Croatian Radio

Just use a url like:

    [http://hrpodcasts.appspot.com/rss?show=SHOW_NAME_HERE](http://hrpodcasts.appspot.com/rss?show=SHOW_NAME_HERE)
